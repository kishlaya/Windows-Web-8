Project Specifics

Features

.

Built using modern HTML5, CSS3 and jQuery framework alognwith support for all modern browsers including Chrome, Firefox and IE. Please switch to full screen mode for best experience.

Brackets is more than awesome

The entire project has been coded in the new and awesome Adobe Brackets 1.3

SASS Preprocessed

CSS3 has been coded via CSS Preprocessor SASS. Also, the project features an intensive use of CSS3 and its features. You will find custom CSS tooltips, modified context-menus, cool transformations, smooth transitions, and a whole bunch of beautiful components.

jQuery completes the magic

Using jQuery to control the entire sequence and to integrate functionality within this project.

Drag n drop!

Uses the jQuery UI Draggable plugin for allowing rearrangement of metro ui tiles and desktop icons and for dragging window applications.

Icons brings it to life!

All the icons (except the Brackets icon) have been served from Ionicons library.

How about smartphones?

All components have been carefully sized to adapt a responsive environment. Therefore, windows web 8 performs as awesome even when you zoom-in zoom-out or even if you are viewing this project on your smartphone or tablet.

Boot Loader

The Pacejs has integrated with the boot screen, so that the boot process only finishes once everything is loaded. That takes the user experience to greater heights.
